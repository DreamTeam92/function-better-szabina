# Project Summary
Function Better is a premium personal brand website focused on luxury movement, strength, and recovery coaching. Located in Costa Blanca South, Spain, it is spearheaded by Sabina, a Biomechanics Specialist with over 23 years of experience. The platform aims to provide an elegant online presence that effectively showcases her premium services, philosophy, qualifications, and client testimonials, catering to affluent clients seeking personalized coaching.

# Project Module Description
The website consists of several functional modules:
- **Navigation**: Responsive navigation bar with scroll detection.
- **Hero Section**: Full-screen layout introducing the brand with refined imagery.
- **Experience Section**: Details the coaching process (Assess → Plan → Coach → Thrive).
- **Services Section**: Displays premium service offerings.
- **Philosophy Section**: Explains the brand's wellness approach.
- **Meet Sabina Section**: Showcases Sabina's qualifications and experience.
- **Client Experience Section**: Features testimonials from clients.
- **Location Section**: Lists service areas across Costa Blanca South.
- **Contact Section**: Inquiry form for potential clients.
- **Footer**: Contains brand information and navigation links.

# Technology Stack
- **React 18**: JavaScript library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for improved development.
- **Vite**: Build tool for faster development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **Fonts**: Cormorant Garamond (headings), Montserrat (body).

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the development server:
   ```bash
   pnpm run dev
   ```
